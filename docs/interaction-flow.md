# Interaction Flow

This is a proposed early interaction flow, not a completed implementation.

## New Container

1. Place empty container on scale.
2. Tap/register blank NFC tag.
3. Save empty weight.
4. Name container.
5. Optionally assign food/category.

## Meal Prep

1. Place container on scale.
2. Fill container.
3. Save starting weight.
4. Set prep date.
5. Optionally set serving count.

## Later Use

1. Place container on scale.
2. Scale reads tag.
3. Scale subtracts empty weight.
4. Interface shows remaining contents.
5. Save new reading to history.
