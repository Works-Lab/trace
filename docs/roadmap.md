# Roadmap

This roadmap is a draft direction, not a release promise.

## v0.1 - Manual Container Registry and Weigh-In

- Register containers manually.
- Save empty weights.
- Record total and contents weight.

## v0.2 - NFC Read/Write Registration

- Assign or write NFC tags.
- Read tags during weigh-ins.
- Connect readings to container records automatically.

## v0.3 - Local Web UI Polish

- Improve registration flow.
- Improve live reading display.
- Make container history easier to inspect.

## v0.4 - Fallback Device AP Mode

- Explore direct-device access point behavior.
- Explore captive portal setup or recovery flow.
- Keep normal home WiFi support as the expected mode.

## v0.5 - History, Servings, Export

- Track reading history.
- Estimate servings remaining.
- Add practical export paths.

## Later - Non-Food Inventory Mode

- Explore supplies, bins, hardware, consumables, and other physical inventory use cases.
- Keep this separate from the first meal-prep loop until the core behavior is useful.
