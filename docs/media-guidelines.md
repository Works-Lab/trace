# Media Guidelines

Document Trace prototypes in a way that makes the hardware and interaction easy to understand later.

For each meaningful hardware revision, capture:

- scale alone
- container on scale
- NFC tag close-up
- phone or local UI
- short video of tag/read/weigh flow if possible
- hardware revision and date in the file name or caption

Do not wait for a fully polished enclosure or interface. Early media is useful when it shows what changed, what worked, and what still needs attention.
